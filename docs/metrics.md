# Performance Metrics

The following metrics will be collected throughout the project.

| Metric | Source | Purpose |
|---------|--------|---------|
| Runtime | gem5 | Overall execution latency |
| Instruction Count | gem5 | Measure execution overhead |
| IPC | gem5 | Pipeline utilization |
| Cache Misses | gem5 | Memory behavior |
| FLOPs | Analytical | Computational work |
| Arithmetic Intensity | Derived | Compute vs memory analysis |

Additional metrics may be introduced as required by future case studies.
